# KTEU Shopify Store Theme 🚀

這是 KTEU Shopify 商店的客製化主題開發專案。本專案結合了 **Shopify CLI** 開發流程與 **GitHub 整合模式**，並串接了 **Lantern Quiz** 與 **Google Sheets** 資料自動化。

## 🛠 開發環境設定

### 1. 必要工具
- [Shopify CLI](https://shopify.dev/docs/themes/tools/cli)
- Git & GitHub Account
- VS Code (推薦編輯器)

### 2. 啟動開發伺服器
在本機進行開發時，請執行以下指令以啟動即時預覽：
```powershell
shopify theme dev --store znsc3u-vt.myshopify.com
